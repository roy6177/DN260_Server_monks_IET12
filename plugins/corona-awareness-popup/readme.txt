== Description ==

Corona Awareness Popup is a very simple plugin used to raise awareness against the deadly Corona (COVID-19) virus spreading throughout the world.
With just a simple editable popup, website owners can show the information related to the symptoms of the disease, and the steps that can be taken to prevent it.
Additionally, a button link can also be added to direct user to the WHO website, or another post/page in the website itself.

Key features include:
* Editable Popup Image
* Editable content with Rich Text editor.
* Customizable button.
* Option to add custom CSS to modify the popup according to the user needs.
* Change the Popup display frequency

Privacy and GDPR
This plugin does not asks or collects the user data.

== Installation ==
1. Upload plugin zip file to the /wp-content/plugins/ directory.
2. Activate the plugin through the Plugins menu in WordPress.
3. Set your options from the Settings -> Corona Awareness admin area.

Alternatively:

1. Login to your WordPress admin area
2. Search for Corona Awareness Popup
3. Click install Corona Awareness Popup
4. Activate through the Plugins menu in WordPress or when asked during installation
5. Set your options from the Settings -> Corona Awareness admin area
